# DjangoPerWeb

##### 环境

python3.9 Django4

##### 准备

```
pip install Django PyMySQL
修改数据库
修改常言用户
```

##### 用户

```
python3 manage.py createsuperuser
```

##### 使用

```
python3 manage.py makemigrations blog
python3 manage.py migrate
python3 manage.py runserver
```
